## Instruçoes

Instale as dependências
`npm install`  
Inicie o projeto **Necessário back-end estar inicializado**
`npm start`

## Controles

**W** - Cima  
**S** - Baixo  
**E** - Seleciona  
**Q** - Volta 

## Apagando Livros

- Selecione *Listar Livros* 
- Selecione o livro a ser  deletado
- Pressione *E* e confirme
