## Instruçoes

Instale as dependências
`npm install`  
Inicie o projeto **Necessário back-end estar inicializado**
`npm start`

## Controles

**W** - Cima  
**S** - Baixo  
**E** - Seleciona  
**Q** - Volta 

## Adicionar livro ao carrinho

- Selecione *Listar Livros* 
- Selecione o livro a ser  adicionado ao carrinho
- Pressione *E* e confirme
